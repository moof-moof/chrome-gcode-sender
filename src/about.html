What is this?

gcode-sender is a chrome application capable of sending gcode commands to a USB gcode intrepreter like Grbl. The source code and documentation for gcode-sender are available on github (https://github.com/sir-buckyball/chrome-gcode-sender).

Actually, this implementation is GeeSaw Feeder, a fork of the above application, which is preferably locally stored and loaded.
